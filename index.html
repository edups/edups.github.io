SideoMatic

How to use SideoMatic Plugin

SideoMatic it's a jQuery plugin to add a beautiful and customizable side menu to your projects.

Follow these steps:

The Markup

    Menu and links
  • Put an unordered list with your links in a "side-menu" class.
    • Content
  • Put your content in a div with the "content" class.
    • One or two menu buttons
  • One or two images, icons, logos what you want, with the "menu" class.
  • - If you choose only one image add the "menu-bar"class. (*See Customize section)
  • - If you choose two images, add the "menu-bar"class for the first image and "menu-left" for the second.
      
        <div class="side-menu">
       <h2 class="title">YourTitleMenu</h2>
	<ul>
		<li><a href="">YourLink01</a></li>
		<li><a href="">YourLink02</a></li>
		<li><a href="">YourLink03</a></li>
	</ul>
	</div>
	
	<div class="content">
	<img class="menu menu-bar"></img>
	<img class="menu menu-left"></img>
	
        //*************ADD YOUR CONTENT HERE*************//
	
	</div> 
	

     
       //CSS
        <script src="css/sideomatic.css">
        <script src="css/sideomatic.min.css">

       
        //jQuery
        <script src="js/jquery.min.js"> </script>


        //SideoMatic js
        <script src="js/sideomatic.js"> </script>
        <script src="js/sideomatic.min.js"> </script>

        //SideoMatic simple call
        $(".side-menu").sideomatic();

CSS and JS

  • Styles
    • Link the sideomatic.css or the sideomatic.min.css

  • Mandatory jQuery
    • Link the jQuery library

  • Don't forget the Js
    • Link the sideomatic.js or the sideomatic.min.js

  • Finally call the Plugin
    • At the end of the body.

Customize your Menu

    Customize the colors
  • By default your menu has black colors, but maybe you want another cool combination.
  • There is no problem, you can customize all the colors of the menu

    • Change the call of the Plugin
  • Simply set the colors in the call of the plugin in this way

    • One button menu
  • By default the plugin works with two buttons, if you want only one, add in the call the value "onlyOne" and set to "true".
      
        //Set your colors
       $(".side-menu").sideomatic({
   		menuColor : 'yellow',
		menuText: '#fff',
		titleColor:'red',
		titleText: 'blue',
		linkBorderColor: '#2a2a2a',
		linkText: 'black',
		buttonColor: 'transparent',
		buttonText:'orange',
		linkOver: 'tomato'
        });
        
        //Set one image button
        $(".side-menu").sideomatic({
   	    onlyOne: false
        });
